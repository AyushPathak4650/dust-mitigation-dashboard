# Security Policy

## Supported Versions
We patch vulnerability bugs for the following supported active project versions:

| Version | Supported |
| --- | --- |
| 1.2.x | Yes |
| < 1.0.x | No |

## Reporting a Vulnerability
We take the security of our application seriously. If you identify any security vulnerability, please report it immediately:

1. **Do not** open a public issue on GitHub.
2. Email reports to: **security@cecb.cg.gov.in** with the subject heading `VULNERABILITY REPORT - AeroDust-Shield`.
3. Include details:
   * Description of the vulnerability.
   * Steps to reproduce (e.g. proof of concept payloads or execution script).
   * Visual evidence (screenshots/recordings).
   * Potential impact details.

We will acknowledge receipt within 48 hours and work to patch the issue within 14 days. We appreciate your assistance in keeping this project secure.
